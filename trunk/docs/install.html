JavaTM API for XML Processing
Installation

Version: 1.1ea2 (Early Access 2)

System Requirements

This version of the Java API for XML Processing requires:
  • JDK 1.1.8 or later (including any version of Java 2)
  • At least 8MB of free disk space

Installation

  1. Download the zip file.
  2. Unzip and install the file.

    You may use the jar command from the JDK or Java 2 SDK to unzip the distribution or use another platform specific utility to do this. Unzipping the file creates a directory of the form jaxp-<version> within the extraction directory.

  3. Update the java classpath.

    You can either update the CLASSPATH environment variable to include the three jar files, or you can install the jar files as an extension to Java 2.

    To update the CLASSPATH variable, add the following components. Let $XML represent the directory where you installed JAXP.

    • On Unix:
      $XML/jaxp.jar:$XML/crimson.jar:$XML/xalan.jar
    • On Win32:
      $XML\jaxp.jar;$XML\crimson.jar;$XML\xalan.jar

    To install these jar files as an extension of Java 2, copy them to the $JAVA_HOME/jre/lib/ext directory. (The JAVA_HOME directory is the one that contains bin/java.) This eliminates the need to modify class paths. (This technique may also be used with the Java Plug-in.)

  4. Run the sample programs. Please refer to information on examples.
